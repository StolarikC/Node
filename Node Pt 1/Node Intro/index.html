Intro to Node

What is node?

Node is a JavaScript runtime that executes code outside of the browser.

We can use the same JS syntax we know to write server-side code, instead of relying on other languages like Python or Ruby.


What to build with node?

  • Web Servers
  • Command Line Tools
  • Native Apps (VSCode is a node app!)
  • Video Games
  • Drone Software
  • Much Much More!

The Node REPL

Read. Evaluate. Print. Loop.

Similar to the "console" used in browser-based JS.

  1. Type "node" to enter the REPL.
  2. Enter desired jsnode content.
  3. type ".exit" to exit the REPL.

We can call "node nameOfJavaScriptFile" to execute it in the terminal as nodeJS.

IF the file is NOT in your current directory, you'll need to reference it correctly to call it from elsewhere, but it can be done.
